
### **English Documentation: How to Use cPanel File Manager**

This video explains how to use **cPanel's File Manager** and how it helps in creating and managing a website by managing website files.

---

#### **How Does cPanel File Manager Work?**

The **File Manager** in cPanel is a useful tool that allows users to easily manage files in their hosting space. When you want to create a website, you need to use the File Manager to upload, edit, and manage your website files.

**Public_HTML Folder:**

There is a folder in cPanel called **public_html**, which is the root folder of your website. To create a website, you must store your website's coding files inside this folder. For example, if you create an **index.html** file, it must be placed inside the **public_html** folder.

**Editing Files and Adding CSS:**

You can edit files in the File Manager. For example, if you write a simple **h1 tag** in **index.html**, you can then add CSS to style it. Example:

```html
<h1>I am Minhaj</h1>
```

Then, add CSS for styling:

```html
<style>
  h1 {
    color: red;
    font-size: 50px;
    padding: 100px;
  }
</style>
```

In this way, you can customize the design of your website.

**Installing WordPress:**

To install WordPress, files like **wp-admin**, **wp-content**, **wp-includes**, and **index.php** will be uploaded in the File Manager. After installation, you can customize the WordPress theme and change your website's design.

**Key Features of File Manager:**

* Uploading and managing website files
* Editing HTML, PHP files
* Adding CSS and customizing the design
* Installing and managing WordPress

---

### **Key Takeaways**

* The **public_html** folder is where all your website's files should go.
* You can edit HTML files and add custom CSS through the File Manager.
* WordPress installation via the File Manager brings core WordPress files into the system, enabling you to customize the theme and website design.

#### **সি প্যানেলের ফাইল ম্যানেজার কীভাবে কাজ করে?**

সি প্যানেলের ফাইল ম্যানেজার একটি গুরুত্বপূর্ণ টুল যা ব্যবহারকারীদের তাদের হোস্টিং স্পেসে ফাইলগুলো সহজে ম্যানেজ করতে সাহায্য করে। যখন আপনি একটি ওয়েবসাইট তৈরি করতে চান, তখন ফাইল ম্যানেজারটি ব্যবহার করে ওয়েবসাইটের ফাইলগুলো আপলোড, এডিট, এবং ম্যানেজ করা যায়।

**পাবলিক_HTML ফোল্ডার:**

সি প্যানেলে একটি ফোল্ডার রয়েছে, যাকে **পাবলিক_এইচটিএমএল** বলা হয়, যা আপনার ওয়েবসাইটের রুট ফোল্ডার। ওয়েবসাইট তৈরি করতে হলে এই ফোল্ডারের ভিতরে ওয়েবসাইটের কোডিং ফাইলগুলো রাখতে হয়। উদাহরণস্বরূপ, যদি আপনি একটি **ইন্ডেক্স.html** ফাইল তৈরি করেন, তাহলে এই ফাইলটি পাবলিক_এইচটিএমএল ফোল্ডারে রাখতে হবে।

**ফাইল এডিট এবং সিএসএস অ্যাড করা:**

ফাইল ম্যানেজারে ফাইলগুলো এডিট করতে পারেন। উদাহরণস্বরূপ, আপনি **ইন্ডেক্স.html** ফাইলে একটি সাধারণ **এইচ1 ট্যাগ** লিখে, তারপর সিএসএস ব্যবহার করে স্টাইল অ্যাড করতে পারেন। যেমন:

```html
<h1>আই অ্যাম মিনহাজ</h1>
```

এবং সিএসএস স্টাইল যোগ করতে:

```html
<style>
  h1 {
    color: red;
    font-size: 50px;
    padding: 100px;
  }
</style>
```

এইভাবে আপনি আপনার ওয়েবসাইটের ডিজাইন কাস্টমাইজ করতে পারেন।

**ওয়ার্ডপ্রেস ইনস্টলেশন:**

ওয়ার্ডপ্রেস ইনস্টল করতে হলে, ফাইল ম্যানেজারে কিছু ফাইল আপলোড হবে, যেমন **wp-admin**, **wp-content**, **wp-includes**, এবং **index.php**। এর পরে, আপনি ওয়ার্ডপ্রেসের থিম কাস্টমাইজ করতে পারবেন এবং আপনার ওয়েবসাইটের ডিজাইন পরিবর্তন করতে পারবেন।

**ফাইল ম্যানেজারের গুরুত্বপূর্ণ ফিচারস:**

* ওয়েবসাইটের ফাইল আপলোড এবং ম্যানেজমেন্ট
* HTML, PHP ফাইল এডিট করা

### Summary of the Video Content: Understanding cPanel File Manager and WordPress Installation

The video provides a detailed walkthrough of how to use the **cPanel File Manager** for managing website files, especially focusing on the **public_html** directory and the process of installing WordPress.

---

### Key Concepts and Workflow

- **cPanel Overview:**
  - cPanel is a hosting control panel where server space is allocated for hosting websites.
  - The **File Manager** is a key tool within cPanel to manage website files and folders.

- **public_html Folder:**
  - This is the **root folder** where all website files must be stored for the domain.
  - When a user visits the domain, files inside this folder are served, such as HTML, PHP, or WordPress files.

- **Creating and Editing Files:**
  - Users can create files like `index.html` or `index.php` in the **public_html** folder.
  - Editing a file (e.g., `index.html`) can be done by clicking the **Edit** button in File Manager.
  - HTML content can be added, such as headings (`<h1>` tag) with simple text.
  - CSS styling can be applied inside `<style>` tags to modify appearance:
    - Text color (e.g., red)
    - Font size (e.g., 50px)
    - Padding (e.g., 100px)
    - Background color can also be added for styling.

- **Uploading Full Website Folders:**
  - Entire folders with website code can be uploaded to **public_html**.
  - The main entry file (like `index.html` or `index.php`) must reside inside **public_html** to be accessible.

---

### WordPress Installation Process

- The video briefly demonstrates installing WordPress through an **automatic script** within cPanel:
  - Clicking **Install Now** initiates the WordPress installation at the root (`public_html`).
  - After installation, WordPress core files appear in File Manager inside **public_html**:
    - Key folders: `wp-admin`, `wp-content`, `wp-includes`
    - Important file: `index.php`
  - These files constitute the WordPress website backend.

- WordPress installation files can also be downloaded manually from the official source and uploaded, but the video focuses on the automated script method.

- After installation, visiting the website URL displays a default WordPress theme design.
  - This theme can be customized through the WordPress admin panel.

---

### Important Highlights

- **public_html** is the critical directory for hosting web files.
- File Manager allows both file creation and editing directly on the server.
- Styling HTML via embedded CSS is straightforward in the File Manager editor.
- WordPress files automatically populate the root after installation.
- The default WordPress theme appears immediately after installation, allowing easy customization.

### প্রধান ধারণা ও কাজের ধারা

* **সি প্যানেল পরিচিতি:**

  * সি প্যানেল হল একটি হোস্টিং কন্ট্রোল প্যানেল যেখানে সার্ভার স্পেস হোস্টিংয়ের জন্য বরাদ্দ করা হয়।
  * **ফাইল ম্যানেজার** সি প্যানেলের একটি গুরুত্বপূর্ণ টুল, যা ওয়েবসাইটের ফাইল ও ফোল্ডার ম্যানেজ করার জন্য ব্যবহৃত হয়।

* **public_html ফোল্ডার:**

  * এটি হলো **রুট ফোল্ডার**, যেখানে সমস্ত ওয়েবসাইটের ফাইল রাখতে হয়, যাতে ডোমেইন ভিজিট করার সময় ওয়েবসাইটের ফাইলগুলো দেখা যায়।
  * যখন একটি ব্যবহারকারী ডোমেইনে প্রবেশ করে, তখন **public_html** ফোল্ডারের ভিতরের ফাইলগুলো যেমন HTML, PHP বা ওয়ার্ডপ্রেস ফাইলগুলি সার্ভ করা হয়।

* **ফাইল তৈরি এবং এডিট করা:**

  * ব্যবহারকারীরা **public_html** ফোল্ডারে `index.html` বা `index.php` ফাইল তৈরি করতে পারেন।
  * ফাইলটি এডিট করতে ফাইল ম্যানেজারের **এডিট** বাটনে ক্লিক করা হয়।
  * HTML কনটেন্ট যোগ করা যেমন: `<h1>` ট্যাগ ব্যবহার করে সিম্পল লেখা লেখা যেতে পারে।
  * CSS স্টাইলিংও অ্যাড করা যেতে পারে:

    * টেক্সট কালার (যেমন, লাল)
    * ফন্ট সাইজ (যেমন, 50px)
    * প্যাডিং (যেমন, 100px)
    * ব্যাকগ্রাউন্ড কালারও যোগ করা যায়।

* **সম্পূর্ণ ওয়েবসাইট ফোল্ডার আপলোড করা:**

  * ওয়েবসাইটের কোডসহ পুরো ফোল্ডার **public_html** ফোল্ডারে আপলোড করা যেতে পারে।
  * মূল এন্ট্রি ফাইল যেমন `index.html` বা `index.php` অবশ্যই **public_html** ফোল্ডারে থাকতে হবে, যাতে ওয়েবসাইটটি অ্যাক্সেস করা যায়।

  * ইনস্টলেশনের পর, ফাইল ম্যানেজারে **public_html** ফোল্ডারে ওয়ার্ডপ্রেসের কোর ফাইলগুলো দেখা যায়:

* ইনস্টলেশনের পর, ওয়েবসাইট ইউআরএল ভিজিট করলে একটি ডিফল্ট ওয়ার্ডপ্রেস থিম ডিজাইন দেখা যাবে।

  * এই থিমটি ওয়ার্ডপ্রেস অ্যাডমিন প্যানেল থেকে কাস্টমাইজ করা যেতে পারে।

### প্রধান অন্তর্দৃষ্টি

* **public_html** এর স্ট্রাকচার এবং এর ভূমিকা বুঝে ওয়েবসাইট ডিপ্লয়মেন্ট সহজ হবে।
* **ফাইল ম্যানেজার** সরাসরি সার্ভারে ফাইল ম্যানেজ করার সুযোগ দেয়, যার ফলে FTP প্রয়োজন হয় না।
* **ওয়ার্ডপ্রেস ইনস্টলেশন** রুট ডিরেক্টরিতে কোর ফাইলগুলো ইন্টিগ্রেট করে, যা সহজ ওয়েবসাইট সেটআপের জন্য সহায়ক।
* **HTML এবং CSS ডিরেক্ট এডিটিং** ফাইল ম্যানেজারে দ্রুত ওয়েবসাইট পরিবর্তন করতে সহায়তা করে।
* **অটোমেটেড ওয়ার্ডপ্রেস ইনস্টলেশন স্ক্রিপ্ট** ম্যানুয়াল ইনস্টলেশনের তুলনায় সহজ এবং দ্রুত।

### উপসংহার

এই ভিডিওটি সি প্যানেল ফাইল ম্যানেজারের মৌলিক ব্যবহারের ধারণা প্রদান করে, বিশেষ করে **public_html** ডিরেক্টরির মধ্যে কিভাবে ফাইল তৈরি এবং এডিট করতে হয় তা ব্যাখ্যা করা হয়েছে। এটি সিম্পল HTML পেজ তৈরি এবং ওয়ার্ডপ্রেস ইনস্টলেশন প্রক্রিয়া প্রদর্শন করে, যা ওয়েবসাইট পরিচালনা ও কাস্টমাইজ করার জন্য একটি শক্তিশালী ভিত্তি প্রদান করে। এই মৌলিক জ্ঞান যেকোনো ব্যক্তি জন্য গুরুত্বপূর্ণ যারা সি প্যানেল ব্যবহার করে ওয়েবসাইট পরিচালনা করছেন।
