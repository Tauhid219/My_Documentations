PHP-তে **Named Arguments** ফাংশন কল করার সময় প্যারামিটারগুলোর নাম ব্যবহার করে মান পাস করার একটি বৈশিষ্ট্য। PHP 8.0 থেকে এটি চালু হয়েছে এবং এটি ফাংশন কলকে আরও স্পষ্ট এবং পাঠযোগ্য করে তোলে, বিশেষ করে যখন ফাংশনে অনেক প্যারামিটার থাকে।

### 1. **Named Arguments কী?**
Named Arguments হল একটি বৈশিষ্ট্য যা আপনাকে ফাংশন কলের সময় প্যারামিটারগুলোর নাম উল্লেখ করে মান পাস করার সুবিধা দেয়। এর ফলে, আপনি প্যারামিটারগুলোর অবস্থান নির্ধারণ করার প্রয়োজন ছাড়াই শুধুমাত্র নাম ব্যবহার করে মান পাস করতে পারেন।

### 2. **Named Arguments ব্যবহার করা:**
নাম দিয়ে প্যারামিটার পাস করার জন্য প্যারামিটারটির নাম এবং মানের মধ্যে `=` চিহ্ন ব্যবহার করতে হয়।

#### উদাহরণ:
```php
<?php
function greet($name, $age = 18, $city = "Dhaka") {
    echo "Hello, $name! You are $age years old and live in $city.";
}

// Named Arguments ব্যবহার
greet(name: "Alice", city: "Chittagong", age: 25); 
// আউটপুট হবে "Hello, Alice! You are 25 years old and live in Chittagong."
?>
```

এখানে:
- `name`, `age`, এবং `city` প্যারামিটারগুলো নাম সহ পাস করা হয়েছে। 
- আপনি প্যারামিটারগুলোর অবস্থান অনুসরণ না করে নামের মাধ্যমে তাদের মান নির্ধারণ করতে পারবেন।

### 3. **Named Arguments এর সুবিধা:**
- **প্যারামিটারগুলোর অবস্থান নির্ভরতা দূর হয়**: আপনি যে কোন প্যারামিটার পাস করতে পারেন এবং তাদের অবস্থান নিয়ে চিন্তা করতে হবে না।
- **কোডের পাঠযোগ্যতা বৃদ্ধি**: নাম ব্যবহার করে ফাংশন কল করলে এটি আরো পরিষ্কার এবং বুঝতে সহজ হয়।
- **ডিফল্ট মান ব্যবহার করা সহজ**: যেসব প্যারামিটার ডিফল্ট মান সহ থাকে, তাদের আপনি পাস না করলেও সমস্যা হবে না। তবে, নাম সহ পাস করলে কোড আরো পরিষ্কার হবে।

### 4. **Named Arguments-এর কিছু অন্যান্য উদাহরণ:**

#### 1. **কেবল নির্দিষ্ট প্যারামিটার পাস করা:**
```php
<?php
function add($a, $b = 5, $c = 10) {
    return $a + $b + $c;
}

echo add(a: 3, c: 7); // আউটপুট হবে 15 (3 + 5 + 7)
?>
```
এখানে:
- `b` প্যারামিটারটি ডিফল্ট মান `5` নিয়ে এসেছে, এবং `a` ও `c` প্যারামিটারগুলো নাম দিয়ে পাস করা হয়েছে।

#### 2. **একাধিক প্যারামিটার নির্দিষ্ট করা:**
```php
<?php
function createUser($username, $email, $role = 'user', $status = 'active') {
    echo "Username: $username, Email: $email, Role: $role, Status: $status";
}

createUser(username: "john_doe", email: "john@example.com", status: "inactive"); 
// আউটপুট হবে "Username: john_doe, Email: john@example.com, Role: user, Status: inactive"
?>
```

এখানে:
- `role` প্যারামিটারটি ডিফল্ট মান `"user"` নিয়েছে, কারণ সেটি নাম সহ পাস করা হয়নি।

### 5. **Named Arguments এর সীমাবদ্ধতা:**
- Named arguments শুধুমাত্র **ফাংশন ডিক্লারেশন** এর প্যারামিটারগুলির জন্য কাজ করবে, এবং এগুলি **নামযুক্ত প্যারামিটার** হিসেবে কাজ করবে যেগুলোর **অবশ্যই ডিফল্ট মান থাকতে হবে** অথবা আপনি তাদের **সঠিকভাবে কল করবেন**।
- এটি শুধুমাত্র **ফাংশন কল** এর সময় কাজ করে, প্যারামিটার ডিক্লারেশনের সময় নয়।

### সারাংশ:
- **Named Arguments** আপনাকে প্যারামিটারগুলোর নাম ব্যবহার করে মান পাস করতে দেয়, যা কোডের পাঠযোগ্যতা বাড়ায় এবং প্যারামিটারগুলোর অবস্থান নির্ভরতা কমায়।
- আপনি প্যারামিটারগুলোর নাম এবং মান ব্যবহার করে ফাংশন কল করতে পারবেন, এবং এর ফলে কোডে ত্রুটি কমে যায়।
- Named arguments শুধুমাত্র PHP 8.0 এবং তার পরবর্তী সংস্করণে উপলব্ধ।
