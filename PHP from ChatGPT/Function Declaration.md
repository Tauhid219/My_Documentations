PHP-তে **ফাংশন ডিক্লারেশন** হলো ফাংশন তৈরি করার প্রক্রিয়া, যেখানে আপনি ফাংশনের নাম, আর্গুমেন্ট এবং তার রিটার্ন টাইপ (যদি থাকে) ঘোষণা করেন। এর মাধ্যমে PHP জানিয়ে দেয় যে ফাংশনটি কি কাজ করবে এবং এর আর্গুমেন্ট ও রিটার্ন ভ্যালু কেমন হবে।

### 1. **ফাংশন ডিক্লারেশন:**
ফাংশন ডিক্লারেশনে ফাংশনের নাম এবং আর্গুমেন্টগুলি প্রথমে প্রকাশ করা হয়।

```php
<?php
function function_name($arg1, $arg2) {
    // কোড এখানে থাকবে
}
?>
```

এখানে:
- `function` কিওয়ার্ড ফাংশন তৈরির জন্য ব্যবহৃত।
- `function_name` হলো ফাংশনের নাম।
- `$arg1, $arg2` হলো ফাংশনের আর্গুমেন্ট (parameters)।

### 2. **ফাংশন ডিক্লারেশন উদাহরণ:**

```php
<?php
function greet($name) {
    echo "Hello, $name!";
}

greet("John"); // আউটপুট হবে "Hello, John!"
?>
```

এখানে `greet` হলো ফাংশনের নাম, এবং `$name` হলো একমাত্র আর্গুমেন্ট যা ব্যবহারকারীর নাম ধরে ফাংশনে পাস করা হয়।

### 3. **ফাংশন ডিক্লারেশনে ডিফল্ট আর্গুমেন্ট:**
ফাংশন ডিক্লারেশনে আপনি আর্গুমেন্টগুলোর জন্য ডিফল্ট মানও নির্ধারণ করতে পারেন। এটি ফাংশন কল করার সময় যদি কোনো আর্গুমেন্ট না দেওয়া হয় তবে ডিফল্ট মান ব্যবহার হবে।

```php
<?php
function greet($name = "Guest") {
    echo "Hello, $name!";
}

greet(); // আউটপুট হবে "Hello, Guest!"
greet("Alice"); // আউটপুট হবে "Hello, Alice!"
?>
```

### 4. **ফাংশন রিটার্ন টাইপ ডিক্লারেশন:**
PHP 7 এবং তার পরবর্তী ভার্সনে, আপনি একটি ফাংশনের রিটার্ন টাইপও ডিক্লেয়ার করতে পারেন। এটি ফাংশনটির দ্বারা ফেরত দেওয়া মানের ধরন স্পষ্ট করে।

```php
<?php
function sum(int $a, int $b): int {
    return $a + $b;
}

echo sum(5, 3); // আউটপুট হবে 8
?>
```

এখানে:
- `int $a, int $b`: ফাংশনের আর্গুমেন্ট হিসেবে দুটি পূর্ণসংখ্যা (integer) নেওয়া হচ্ছে।
- `: int`: রিটার্ন টাইপ স্পেসিফাই করছে যে ফাংশনটি একটি পূর্ণসংখ্যা ফেরত দিবে।

### 5. **ফাংশন ডিক্লারেশন সম্পূর্ণ উদাহরণ:**

```php
<?php
function multiply(float $a, float $b): float {
    return $a * $b;
}

$result = multiply(4.5, 2.3);
echo $result; // আউটপুট হবে 10.35
?>
```

এখানে `multiply` ফাংশন দুটি দশমিক সংখ্যা (float) নেয় এবং তাদের গুণফল (float) রিটার্ন করে।

### সারাংশ:
- ফাংশন ডিক্লারেশন আপনাকে ফাংশনের নাম, আর্গুমেন্ট এবং রিটার্ন টাইপ সঠিকভাবে ঘোষণা করতে সাহায্য করে।
- আপনি ডিফল্ট মান, আর্গুমেন্ট টাইপ, এবং রিটার্ন টাইপ ডিক্লেয়ার করতে পারেন।

এটা যদি পরিষ্কার হয়, তাহলে আপনি আরও কিছু অংশ শিখতে চান বা কোনো বিশেষ কনসেপ্ট নিয়ে আলোচনা করতে চান?
