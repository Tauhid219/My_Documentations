PHP-তে **Default** এবং **Optional Parameters** ফাংশন ডিক্লারেশনের সময় প্যারামিটারগুলোর জন্য একটি ডিফল্ট মান নির্ধারণ করতে ব্যবহৃত হয়, যাতে ফাংশন কলের সময় ঐ প্যারামিটারটি পাস না করা হলে, ডিফল্ট মান ব্যবহার হয়।

### 1. **Default Parameters (ডিফল্ট প্যারামিটার):**
ফাংশনের প্যারামিটারগুলোর জন্য আপনি ডিফল্ট মান নির্ধারণ করতে পারেন। যদি ফাংশন কলের সময় কোনো প্যারামিটার পাস না করা হয়, তবে সেই প্যারামিটার ডিফল্ট মান ব্যবহার করবে।

#### উদাহরণ:
```php
<?php
function greet($name = "Guest") {
    echo "Hello, $name!";
}

greet(); // আউটপুট হবে "Hello, Guest!"
greet("Alice"); // আউটপুট হবে "Hello, Alice!"
?>
```

এখানে `$name` প্যারামিটারটির জন্য ডিফল্ট মান `"Guest"` দেওয়া হয়েছে। যদি কল করার সময় `$name` প্যারামিটারটি পাস না করা হয়, তাহলে `"Guest"` ব্যবহার হবে।

### 2. **Multiple Default Parameters (একাধিক ডিফল্ট প্যারামিটার):**
একাধিক প্যারামিটারও ডিফল্ট মান সহ ব্যবহার করা যেতে পারে। তবে, যেগুলি ডিফল্ট মান পাবে, সেগুলোই কলের সময় পাস না করা হলে ডিফল্ট মান নেবে।

#### উদাহরণ:
```php
<?php
function greet($name = "Guest", $age = 18) {
    echo "Hello, $name! You are $age years old.";
}

greet(); // আউটপুট হবে "Hello, Guest! You are 18 years old."
greet("Alice"); // আউটপুট হবে "Hello, Alice! You are 18 years old."
greet("Bob", 25); // আউটপুট হবে "Hello, Bob! You are 25 years old."
?>
```

এখানে `$name` এবং `$age` দুটি প্যারামিটার রয়েছে, এবং তাদের জন্য ডিফল্ট মান `"Guest"` এবং `18` নির্ধারণ করা হয়েছে। যখন শুধুমাত্র `$name` পাস করা হয়, তখন `$age` প্যারামিটারটি ডিফল্ট মান ব্যবহার করে।

### 3. **Order of Parameters (প্যারামিটারগুলোর অর্ডার):**
যখন আপনি ফাংশনে ডিফল্ট মান নির্ধারণ করেন, তখন ডিফল্ট মান দিয়ে প্যারামিটারগুলো সর্বশেষে রাখতে হবে। কারণ, প্রথমে যদি কোনো প্যারামিটার পাস করা হয় এবং পরবর্তী প্যারামিটারটি ডিফল্ট মান সহ থাকে, তাহলে সেটি সঠিকভাবে কাজ করবে না।

#### ভুল উদাহরণ:
```php
<?php
// এটি কাজ করবে না
function greet($name = "Guest", $age) {
    echo "Hello, $name! You are $age years old.";
}

greet(25); // ত্রুটি হবে
?>
```

এখানে `$age` প্যারামিটারটি ডিফল্ট মান ছাড়া আছে এবং `$name` প্যারামিটারটি ডিফল্ট মান সহ। তাই যদি `$age` পাস না করা হয়, তবে তা একটি ত্রুটি সৃষ্টি করবে।

#### সঠিক উদাহরণ:
```php
<?php
// এটি সঠিক
function greet($name = "Guest", $age = 18) {
    echo "Hello, $name! You are $age years old.";
}

greet(25); // আউটপুট হবে "Hello, 25! You are 18 years old."
?>
```

### 4. **Optional Parameters (অপশনাল প্যারামিটার):**
ফাংশনের প্যারামিটার যদি ডিফল্ট মান সহ থাকে, তবে সেগুলোকে **অপশনাল প্যারামিটার** বলা যায়। এগুলির মান যদি কল করার সময় পাস না করা হয়, তবে ডিফল্ট মান ব্যবহার করা হবে।

```php
<?php
function add($a, $b = 5) {
    return $a + $b;
}

echo add(3);  // আউটপুট হবে 8 (3 + 5)
echo add(3, 7); // আউটপুট হবে 10 (3 + 7)
?>
```

এখানে `$b` প্যারামিটারটি অপশনাল এবং এর ডিফল্ট মান `5`। প্রথম কলের সময় `$b` পাস না করা হলে, ডিফল্ট মান `5` ব্যবহার করা হবে।

### সারাংশ:
- **Default Parameters**: ফাংশন প্যারামিটারগুলোর জন্য ডিফল্ট মান নির্ধারণ করা যেতে পারে, যা ফাংশন কলের সময় পাস না করলে ব্যবহৃত হবে।
- **Optional Parameters**: ডিফল্ট মান সহ প্যারামিটারগুলোকে অপশনাল প্যারামিটার বলা হয়, এবং এগুলো ফাংশন কলের সময় পাস না করলে ডিফল্ট মান ব্যবহার করবে।
- **Order**: ডিফল্ট মান সহ প্যারামিটারগুলো ফাংশন ডিক্লারেশনের শেষে থাকতে হবে, নয়তো ত্রুটি হবে।

এখন যদি কোনো বিশেষ প্রশ্ন থাকে বা আপনি অন্য কিছু শিখতে চান, জানাবেন!
